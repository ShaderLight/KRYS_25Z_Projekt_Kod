# KRYS_25Z_Projekt_Kod

Implementacja ataku Kipnisa Shamira na potrzeby przedmiotu KRYS 25Z.

### Wymagania techniczne

Mocno rekomendowane jest uruchomienie implementacji (`krypto.ipynb`) za pomocą środowiska *Anaconda*.
W przeciwnym wypadku należy samemu skonfigurować środowisko Python będące w stanie uruchamiać notatniki Jupyter.

Po instalacji środowiska należy doinstalować bibliotekę `galois` (oraz `numpy` jeżeli nie korzystamy z *Anaconda*). Można to zrobić w interfejsie `Anaconda` lub w CLI.

Anaconda:
```
conda install galois
```

Python (bez środowiska Anaconda):
```
pip install galois numpy
```

Implementację testowano na wersji języka Python `3.14.2`.
